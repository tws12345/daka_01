    3）python 中 is , is not 与 == , != 的区别是什么？
        1. is, is not 对比的是两个变量的内存地址
        2. ==, != 对比的是两个变量的值
        假如比较的两个变量，指向的都是地址不可变的类型（str等），那么is，is not 和 ==，！= 是完全等价的。
        假如对比的两个变量，指向的是地址可变的类型（list，dict，tuple等），则两者是有区别的
